# Poly Haven 1K Tile Crops

Source: https://polyhaven.com/
License: CC0, as provided by Poly Haven.

This folder contains 1K Diffuse source textures downloaded from Poly Haven and cropped
512x512 PNG tile chunks for Unity use.

Originals:
- `Originals/*_diff_1k.jpg`: downloaded 1K Diffuse maps.

Cropped tiles:
- `CroppedTiles/*_tile.png`: 512x512 crops chosen to keep grout, board seams, or paving joints visible.

Crop notes:
- Square tile materials generally keep a 2x2 or larger visible tile group.
- Diagonal/diamond materials are not rotated; the diagonal layout is preserved.
- Wood materials keep several plank strips plus visible seams.
